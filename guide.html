1. Requirements 1. Configuration requise

  • BBC micro:bit V2 (V1 lacks sound sensor)BBC micro:bit V2 (V1 n'a pas de capteur de son)
  • USB cable or battery packCable USB ou bloc de piles
  • Chrome or Edge browser (Web Bluetooth required)Navigateur Chrome ou Edge (Web Bluetooth requis)
  • Bluetooth enabled on your computerBluetooth active sur votre ordinateur
Important: Firefox and Safari do NOT support Web Bluetooth. Use Chrome or Edge. Important : Firefox et Safari ne supportent PAS le Web Bluetooth. Utilisez Chrome ou Edge.

2. Flash the Firmware 2. Flasher le firmware

The micro:bit needs special firmware to communicate with the app via Bluetooth. Le micro:bit a besoin d'un firmware special pour communiquer avec l'application via Bluetooth.

Method A: From the App (easiest)Methode A : Depuis l'application (le plus simple)

1Open the app and click micro:bit Firmware buttonOuvrez l'application et cliquez sur le bouton micro:bit Firmware
2Click Copy CodeCliquez sur Copy Code
3Click Open MakeCode — it opens makecode.microbit.orgCliquez sur Open MakeCode — cela ouvre makecode.microbit.org
4In MakeCode, create a New ProjectDans MakeCode, creez un Nouveau Projet
5Click Extensions (bottom of sidebar), search bluetooth, add it. Accept removing radio.Cliquez sur Extensions (bas de la barre laterale), cherchez bluetooth, ajoutez-le. Acceptez de retirer la radio.
6Switch to JavaScript mode (top toggle)Passez en mode JavaScript (bouton en haut)
7Click on main.ts in the Explorer panel (bottom-left), select all (Ctrl+A), paste (Ctrl+V)Cliquez sur main.ts dans l'explorateur (en bas a gauche), selectionnez tout (Ctrl+A), collez (Ctrl+V)
8Check: bottom bar should show 0 errorsVerifiez : la barre du bas doit afficher 0 erreurs
9Click Download — a .hex file downloadsCliquez sur Telecharger — un fichier .hex se telecharge
10Connect micro:bit via USB. Drag the .hex file onto the MICROBIT drive in your file managerConnectez le micro:bit en USB. Glissez le fichier .hex sur le lecteur MICROBIT dans votre gestionnaire de fichiers
11Wait for the micro:bit to reboot. It should show an X icon = ready!Attendez le redemarrage du micro:bit. Il doit afficher une icone X = pret !
Critical: You MUST add the Bluetooth extension (step 5) before pasting the code, otherwise MakeCode will show hundreds of errors. Critique : Vous DEVEZ ajouter l'extension Bluetooth (etape 5) avant de coller le code, sinon MakeCode affichera des centaines d'erreurs.
Flash the firmware on all micro:bits before the workshop. Once flashed, students only need to open the app and click Connect. Flashez le firmware sur tous les micro:bits avant l'atelier. Une fois flashe, les eleves n'ont qu'a ouvrir l'application et cliquer sur Connecter.

3. Connect your micro:bit 3. Connecter votre micro:bit

1Make sure the micro:bit is powered on and showing the X iconAssurez-vous que le micro:bit est allume et affiche l'icone X
2Open the app in Chrome or EdgeOuvrez l'application dans Chrome ou Edge
3Click the green Connect buttonCliquez sur le bouton vert Connect
4Select your micro:bit from the Bluetooth popup (look for BBC micro:bit or uBit)Selectionnez votre micro:bit dans la fenetre Bluetooth (cherchez BBC micro:bit ou uBit)
5Click Pair. The micro:bit shows a checkmark = connected!Cliquez sur Associer. Le micro:bit affiche un V = connecte !

4. Tabs Guide 4. Guide des onglets

🎛️ Controls

  • Show a Message — Type text and click Send to scroll it on the micro:bit LED displayShow a Message — Tapez du texte et cliquez sur Send pour le faire defiler sur l'ecran LED du micro:bit
  • Draw on LEDs — Click and drag on the 5x5 grid to draw patterns. Click Send to display them. Use Quick Shapes (Heart, Smile, Tick) for preset patternsDraw on LEDs — Cliquez et glissez sur la grille 5x5 pour dessiner. Cliquez sur Send pour afficher. Utilisez les Quick Shapes (Heart, Smile, Tick) pour des motifs predefined
  • Make Sounds — Play preset buzzer tones (Low, Beep, High, Melody)Make Sounds — Jouez des sons predefined (Low, Beep, High, Melody)

👀 Sensors

  • Real-time data from: temperature, light, sound, accelerometer (X/Y/Z), compass, buttons A/B, touch pins P0/P1/P2, logo touchDonnees en temps reel : temperature, lumiere, son, accelerometre (X/Y/Z), boussole, boutons A/B, broches tactiles P0/P1/P2, touche logo
  • Each sensor has a mini sparkline chart showing recent valuesChaque capteur a un mini graphique montrant les valeurs recentes
  • Calibration section: calibrate compass (tilt game), set accelerometer zero, adjust sound/light baselinesCalibration : calibrer la boussole (jeu d'inclinaison), mettre l'accelerometre a zero, ajuster les references son/lumiere
Compass data only appears after you click Calibrate in the Calibration section. This avoids the auto-calibration prompt on boot. Les donnees de la boussole n'apparaissent qu'apres avoir clique sur Calibrate dans la section Calibration. Cela evite l'invite de calibration automatique au demarrage.

⚙️ Motors

  • Servo 1 (pin P1) and Servo 2 (pin P2) — control angle 0-180 degrees with slidersServo 1 (broche P1) et Servo 2 (broche P2) — controlez l'angle de 0 a 180 degres avec les curseurs
  • Visual gauge shows the current angleLa jauge visuelle montre l'angle actuel
  • OFF button releases the servo (frees the pin for touch sensing)Le bouton OFF relache le servo (libere la broche pour la detection tactile)

🎮 GamePad

  • D-pad (Up/Down/Left/Right) + Fire buttonD-pad (Haut/Bas/Gauche/Droite) + bouton Fire
  • Perfect for driving robots or playing games on the micro:bitIdeal pour piloter des robots ou jouer a des jeux sur le micro:bit
  • Each button sends a command and shows an arrow on the micro:bit LEDChaque bouton envoie une commande et affiche une fleche sur l'ecran LED du micro:bit

📈 Graph

  • 5 graph types: Line, Bar, Scatter, Area, Realtime5 types de graphiques : Ligne, Barres, Nuage, Aire, Temps reel
  • Sensor checkboxes — select which data to plot (Accel X/Y/Z, Compass, Sound, Light, Temp, Touch)Cases capteurs — selectionnez les donnees a tracer (Accel X/Y/Z, Boussole, Son, Lumiere, Temp, Tactile)
  • Simulate — generates demo data (Sine, Random, Ramp) without a micro:bitSimulate — genere des donnees de demo (Sinus, Aleatoire, Rampe) sans micro:bit
  • Record / Replay / Save — capture sessions and export as JSONRecord / Replay / Save — capturez des sessions et exportez en JSON
  • Annotate — add timestamped notes on the chartAnnotate — ajoutez des notes horodatees sur le graphique
  • Export — PNG image or CSV spreadsheet dataExport — image PNG ou donnees CSV pour tableur
Use the Simulate button for demos without hardware. Students can export CSV data for analysis in a spreadsheet. Utilisez le bouton Simulate pour les demos sans materiel. Les eleves peuvent exporter les donnees CSV pour les analyser dans un tableur.

🎲 3D Board

  • 5 interactive 3D models: micro:bit V2, Robot Buggy, Robot Arm, Balance Game, Weather Station5 modeles 3D interactifs : micro:bit V2, Robot Buggy, Bras Robot, Jeu d'Equilibre, Station Meteo
  • Models animate with live sensor data — tilt the micro:bit to see the 3D model moveLes modeles s'animent avec les donnees des capteurs en direct — inclinez le micro:bit pour voir le modele 3D bouger
  • Drag to rotate, scroll to zoomGlissez pour tourner, scrollez pour zoomer
  • Use the dropdown to switch between modelsUtilisez le menu deroulant pour changer de modele

7. Troubleshooting 7. Depannage

micro:bit not found in scannermicro:bit introuvable dans le scanner

  • Check that Bluetooth is ON on your computerVerifiez que le Bluetooth est active sur votre ordinateur
  • Make sure the micro:bit shows the X icon (firmware is running)Assurez-vous que le micro:bit affiche l'icone X (le firmware est en cours d'execution)
  • The micro:bit must NOT be connected to another device/tabLe micro:bit ne doit PAS etre connecte a un autre appareil/onglet
  • Try resetting the micro:bit (press the reset button on the back)Essayez de reinitialiser le micro:bit (appuyez sur le bouton reset a l'arriere)
  • Use Chrome or Edge — other browsers don't support Web BluetoothUtilisez Chrome ou Edge — les autres navigateurs ne supportent pas le Web Bluetooth

MakeCode shows errors when pasting firmwareMakeCode affiche des erreurs en collant le firmware

  • You forgot to add the Bluetooth extension (step 5 in firmware flashing)Vous avez oublie d'ajouter l'extension Bluetooth (etape 5 du flashage)
  • Make sure you paste into main.ts, not pxt.jsonAssurez-vous de coller dans main.ts, pas dans pxt.json
  • Make sure you're in JavaScript mode, not Blocks modeAssurez-vous d'etre en mode JavaScript, pas en mode Blocs

Compass calibration starts automaticallyLa calibration de la boussole demarre automatiquement

  • This was fixed in V8.1. Make sure you have the latest firmware flashedCeci a ete corrige en V8.1. Assurez-vous d'avoir flashe le dernier firmware

8. Teacher Notes 8. Notes pour les enseignants

Learning ObjectivesObjectifs pedagogiques

  • Understand Bluetooth Low Energy communicationComprendre la communication Bluetooth Low Energy
  • Read and interpret sensor data (temperature, light, motion)Lire et interpreter les donnees des capteurs (temperature, lumiere, mouvement)
  • Control actuators (servos, buzzer, LEDs) from a web interfaceControler des actionneurs (servos, buzzer, LEDs) depuis une interface web
  • Data visualization with real-time graphs and CSV exportVisualisation de donnees avec des graphiques en temps reel et export CSV
  • Introduction to 3D modeling driven by sensor dataIntroduction a la modelisation 3D pilotee par les capteurs

  • 🌡️ Weather Station — Graph temperature and light over time, export CSV for spreadsheet analysis🌡️ Station Meteo — Tracez la temperature et la lumiere au fil du temps, exportez en CSV pour analyse tableur
  • 🎯 Balance Game — Tilt the micro:bit to roll the ball to targets in the 3D view🎯 Jeu d'Equilibre — Inclinez le micro:bit pour faire rouler la balle vers les cibles en 3D
  • 🎵 Sound Meter — Monitor noise levels, use graph annotations to mark events🎵 Sonometre — Surveillez les niveaux sonores, utilisez les annotations du graphique pour marquer les evenements
  • 📐 Motion Tracker — Record accelerometer data, export and analyze in a spreadsheet📐 Tracker de Mouvement — Enregistrez les donnees de l'accelerometre, exportez et analysez dans un tableur
  • 🦾 Robot Arm — Control 2 servos, see the 3D arm respond in real time🦾 Bras Robot — Controlez 2 servos, voyez le bras 3D reagir en temps reel
