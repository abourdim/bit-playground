# أصول التسويق — النسخة العربية

مجموعة كاملة من المرئيات والنصوص التسويقية لعميل أو إدراج Etsy ناطق
بالعربية. تم توليد كل شيء من التطبيق أثناء تشغيله بالعربية (عبر
`#lang=ar`)، مع تعليق صوتي TTS عربي (صوت Naayf)، ونصوص مكتوبة
بالعربية الأصلية مع تخطيط RTL كامل.

## المحتويات

### 📸 لقطات الشاشة — `screenshots/` (58 ملف)

التطبيق بالعربية بالكامل، في جميع أوضاع الملتقط: علامات التبويب
القياسية + الاصطناعية، أزواج قبل/بعد، السمات (الأربع)، النماذج
ثلاثية الأبعاد، الشخصيات (المعلم arctic / الطفل neon / المبدع blaze)،
إثبات العمل دون اتصال، تعليقات SVG، وأربع نسب أبعاد (9:16 / 1:1 /
2:3 / 16:9) لكل علامة تبويب رئيسية.

لكل PNG ملف `.alt.txt` مرافق يحتوي على نص بديل Etsy مُحسَّن لـ SEO
باللغة العربية.

### 🎯 صور hero لـ Etsy — `heroes/` (3 ملفات، 1500×1500)

- `hero-ar-teacher.png` — زاوية المعلم، سمة arctic، لكنة زرقاء
- `hero-ar-kid.png` — زاوية الطفل، سمة neon، لكنة خضراء
- `hero-ar-maker.png` — زاوية المبدع، سمة blaze، لكنة برتقالية

للرفع في الفتحات 1 و9 وشعار Etsy الدوار.

### 🎬 فيديو قصير — `theme-morph.mp4` + `.gif`

6 ثوانٍ بنسبة 1:1 مربعة، تلاشٍ متقاطع بين السمات الأربع بواجهة
عربية. لدبابيس Pinterest ولفائف Instagram.

### 📱 GIFs عرض — `gifs/`

- `ledmatrix-heart.gif` — رسم تدريجي لقلب على مصفوفة المصابيح
- `graph-record.gif` — رسم بياني للمستشعرات في وضع المحاكاة
- `3d-tilt-sweep.gif` — مسح ميل ثلاثي الأبعاد
- `theme-swap.gif` — دوران السمات الأربع

كل ملف بصيغة MP4 + GIF.

### ⚡ إثبات السرعة — `speed-test/`

- `speed-test.mp4` + `.gif` — عرض "3.2 ثانية إلى بيانات حية" مع
  تراكبات عربية (بدء ← جارٍ الاتصال ← ✓ بيانات حية). الحجة التي
  تزيل شك المعلمين.

### ✍️ بنك التسميات A/B — `captions/captions.md`

5 عناوين hero × 5 عناوين فرعية × 5 تسميات اجتماعية × 5 تسميات
Pinterest × 5 عناوين بريد إلكتروني، لكل من الشخصيات الثلاث.
للاختيار منها حسب اختبارات Etsy الأسبوعية.

### ♿ حزمة إمكانية الوصول — `accessibility/`

- `audio-descriptions/*.wav` — 6 أوصاف صوتية مدتها 30 ثانية (صوت
  Naayf ar-SA) للطلاب ضعاف البصر
- `audio-descriptions/*.txt` — نصوص مكتوبة لقراء DAISY
- `braille/` — ملف `printable.html` فقط

> **ملاحظة حول Braille:** الطباعة البارزة العربية تستخدم نظام تعيين
> أحرف مختلف تماماً عن الإنجليزية. لا يتم توليدها تلقائياً من هذا
> النصّ لأنها تتطلب ناسخاً معتمداً (مثلاً: جمعية المكفوفين السعودية).
> الأوصاف الصوتية أعلاه تغطي حالة الاستخدام الأساسية.

### 🖨️ ملصق وبروشور — `print/`

- `poster-a3.pdf` / `.png` — ملصق A3 عمودي (297×420 مم) جاهز للطباعة،
  بتخطيط RTL كامل. علّقه في قاعات STEM والفاب لاب.
- `flyer-a4.pdf` / `.png` — بروشور A4 عمودي (210×297 مم) للمعلمين.
  العرض + خطوات البدء في 60 ثانية + رمز QR.

رمز QR موسوم بـ `utm_medium=print` لتتبع حركة الزيارات من خلال
إحصائيات Etsy.

### 📱 طقم وسائل التواصل الاجتماعي — `social/`

- `ig-post.png` (1080×1080) — منشور Instagram / فيسبوك المربّع

- `twitter.png` (1200×675) — Twitter / X
- `linkedin.png` (1200×627) — LinkedIn

### 🪪 الهوية — `identity/`

- `business-card.pdf` / `.png` — بطاقة عمل 85×55 مم جاهزة للطباعة

### 💬 روبوت الدردشة قبل البيع — `chatbot/`

- `embed.js` (~6 كيلوبايت) — أداة لدمجها في صفحة العرض التوضيحي

  التثبيت، micro:bit V2، العمل دون اتصال، ترخيص المدرسة، إلخ.)
- `embed.html` — مقتطف الكود للدمج

## الأصول المشتركة

هذه الملفات في `output/shared/` (بدون لغة) وتنطبق أيضاً على الإدراج
العربي:

- `ble-dialog/` — مربع حوار Chrome BLE المركّب
- `photoreal/` — عرض ثلاثي الأبعاد واقعي للوحة micro:bit
- `datasets/` — 100 ملف CSV جلسات مستشعرات + README للمعلم
- `stand/` — ملفات SVG + DXF لحامل اكريليك

## إعادة التوليد

إعادة تشغيل سلسلة العربية بالكامل بأمر واحد:

```bash
node etsy-package/tools/build-localized.mjs ar
```

أو أداة فردية:

```bash
node etsy-package/tools/capture-screenshots.mjs --lang ar
node etsy-package/tools/theme-morph.mjs --lang ar
# إلخ.
```

## أين تستخدم

راجع `etsy-package/MARKETING-GUIDE.md` (بالإنجليزية) للتطابق الكامل
بين الفتحات Etsy/Pinterest/YouTube/Instagram. التوصيات تنطبق مباشرة:
استبدل فقط `output/<ملف>` بـ `output/ar/<ملف>` للإدراج الناطق
بالعربية.
